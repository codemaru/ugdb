'Unity 디자이너스 바이블' 샘플 프로그램

Part 4／Section 01 '노드 베이스 셰이더 에디터: ShaderGraph'
===========================================

■ 폴더/파일 설명 
  - \Assets\Scenes\SampleScene.unity
    샘플 기본 씬 파일입니다. 이 파일을 더블 클릭해서 샘플을 열 수 있습니다. '기본적인 샘플 셰이더 만들기' 폴더와 '물결 애니메이션 만들기' 플더로 나누어 각각 샘플로 만든 셰이더가 들어 있습니다.
      
■ 샘플 사용 방법
  본문의 섹션에서 만드는 방법을 소개하고 있는 완성된 셰이더와 완성에 필요한 소재가 들어있습니다.
  각 폴더 안에 'sample.shadergraph' 또는 'water.shadergraph' 파일이 들어있으므로, Unity 상의 Inspector 웬도우에서 'Open Shader Editor' 버튼을 클릭해 수정 모드에 넣어 주십시오.

■ 샘플 파일 이용시 주의점
  제공하는 데이터는 Unity 학습을 위해 만든 것으로, 이 책의 독자분들이 셰이더 만들기에 참고하기 위해 이용하는 용도로만 허가합니다.
