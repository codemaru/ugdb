◆同階層ファイルの説明
・CreatingDocuments.txt
テキストアセットを作る際に必要な文字をピックアップしたデータです。

・ReferenceDocuments.txt
テキストアセットを作る際に参考なったサイトのURLを記載しております。

・Readme.txt
このファイルです。


◆テキストアセットの作成について
CreatingDocuments.txt は ReferenceDocuments.txt に記載されているサイトを参考に、
サンプルで必要になりそうな文字をピックアップしました。

TextMesh Pro の TextAsset を作成する際は、
CreatingDocuments の内容をコピーし、Custom Character Listに貼り付けして利用します。

URLにも記載されていない文字は、発見し次第 CreatingDocuments.txt の
"self additional"以下に追記しております。
