===========================================

■ 파일/폴더 설명
  - SchoolProduction_Readme.txt
    이 파일입니다.

  - SchoolProduction.unitypackage
    Project 윈도우에서 마우스 오른쪽 버튼을 클릭한 뒤, Import Package에서 이 파일을 임포트합니다.
    SchoolProduction_sample 폴더 바로 아래, 세 개의 씬이 포함되어 있습니다.
    
    - PostProcessing_sample: 포스트 프로세스 샘플 씬입니다.
    - Cinemachine_practice: Cinemachine의 간단한 동작 확인용 샘플 씬입니다.
    - Cinemachine_sample: Cinemachine의 다양한 카메라 확인용 샘플 씬입니다.

  ※ 이 샘플 프로젝트를 임포트하기 전에 'Window→Package Manager'에서 'Post Process'와 'Cinemachine'을 설치해야 합니다.
  
  ※ 'Post Process' 샘플 씬에서는 리니어 색공간을 사용합니다. 'Edit→Project Settings'의 Player 카테고리를 선택하고, Other Settings에서 Rendering 내의 Color Space를 'Linear'로 변경해 주십시오.

■ 샘플 파일 이용시 주의점
  제공하는 데이터는 Unity 학습을 위해 만든 것이며, 이 책의 독자가 'Post Processing', 'Cinemachine' 활용에 참고하기 위한 목적으로 이용하는 것만 허가합니다.
