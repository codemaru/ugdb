Part 3／Section 04 「TextMesh Pro: 풍부한 텍스트 표현으로 매력적인 화면 디자인하기'
===========================================

■ 폴더/파일 설명
  - TextMeshProDemo
    TextMeshPro 샘플 Unity 프로젝트폴더입니다. Unity에서 프로젝트를 열 때는 '프로젝트 열기 방법.png' 파일을 참조해 주십시오.

  - TextMeshProDemo_apk/TextMeshProDemo.apk
    TextMeshPro 샘플 안드로이드 애플리케이션입니다.

  - TextMeshProDemo_exe/TextMeshProDemo.exe
    TextMeshPro 샘플 실행 프로그램입니다.

  - Readme.txt
    이 파일입니다.

■ 샘플 애플리케이션 사용 방법
  - 개요
    TextMesh Pro의 기능 설명이나 표시 예시를 목록화 한 애플리케이션입니다. 이 애플리케이션에 표시된 기능들을 구구현합니다.

  - 사용 방법
    1. 화면 상단 4개 메뉴 버튼을 탭하면 각 화면으로 이동합니다.
    2. 'Fonts' 화면에는 TextMesh Pro를 사용한 연출 예시를 표시합니다.
      다양한 폰트를 사용한 텍스트의 분위를 감상할 수 있습니다.
    3. 'RichText' 화면에는 사용할 수 있는 리치 텍스트 목록을 확인할 수 있습니다.
      목록을 탭하면 리치 텍스트 적용 전 텍스트도 확인할 수 있습니다.
    4. 'Inspetor' 화면에는 인스펙터에서 조작할 수 있는 기능 목록을 확인할 수 있습니다.
      리스트를 탭하면. 기능의 상세한 설명이나 표시 예시를 확인할 수 있습니다.
    5. 'Material' 화면에는 머티리얼에서 조작할 수 있는 기능 목록을 확인할 수 있습니다.
      리스트를 탭하면, 기능의 상세한 설명이나 표시 예시를 확인할 수 있습니다.
